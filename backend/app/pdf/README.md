# Шаблоны дизайна PDF

Система шаблонов для генерации профессиональных PDF каталогов десертов.

## Доступные шаблоны

### 1. Minimal (Минималистичный)
**ID:** `minimal`

Чистый и простой дизайн с акцентом на содержании. Идеален для деловых презентаций и технических каталогов.

**Особенности:**
- Минималистичная типографика
- Четкая структура информации
- Нейтральная цветовая палитра
- Максимальная читаемость

### 2. Classic (Классический)
**ID:** `classic`

Элегантный стиль с декоративными элементами. Подходит для традиционных компаний и премиальных брендов.

**Особенности:**
- Декоративные линии и рамки
- Категории в стилизованных блоках
- Классическая типографика
- Сдержанная цветовая палитра

### 3. Modern (Современный)
**ID:** `modern`

Яркий дизайн с цветными акцентами. Отлично подходит для современных брендов и молодежной аудитории.

**Особенности:**
- Яркие цветные полосы
- Акцентные цвета (синий)
- Динамичная компоновка
- Современная типографика

### 4. Luxury (Премиум)
**ID:** `luxury`

Роскошный стиль с золотыми акцентами. Идеален для премиальных продуктов и VIP-клиентов.

**Особенности:**
- Золотые декоративные элементы
- Премиальная типографика
- Элегантные рамки
- Роскошная цветовая палитра

## Использование

Шаблон выбирается при экспорте PDF через параметр `template` в настройках экспорта:

```python
settings = PDFExportSettings(
    dessert_ids=[1, 2, 3],
    template='modern',  # Выбор шаблона
    ...
)
```

## Расширение системы

Для добавления нового шаблона:

1. Создайте класс, наследующийся от `PDFTemplate`
2. Реализуйте методы:
   - `create_styles()` - создание стилей
   - `create_title_page()` - титульный лист
   - `create_dessert_page()` - страница десерта
3. Зарегистрируйте шаблон в словаре `TEMPLATES` в `templates.py`

Пример:

```python
class CustomTemplate(PDFTemplate):
    def create_styles(self):
        # Ваши стили
        pass
    
    def create_title_page(self, story):
        # Ваш титульный лист
        pass
    
    def create_dessert_page(self, story, dessert, styles):
        # Ваша страница десерта
        pass

TEMPLATES['custom'] = CustomTemplate
```
